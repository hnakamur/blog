ambiguous width cjk patchを当てたhomebrew用tmux

2015-12-29

ほぼHomebrewでサクッとpatchを当てる - Qiitaのそのままですが、tmuxのバージョンを2.1に上げたものをhnakamur/homebrew-customに置きました。

パッチ適用版tmuxのインストール

brew tap hnakamur/custom
brew install tmux-patched

tmux.confへの設定追加

Homebrewでサクッとpatchを当てる - Qiitaのコメントにある通り、 ~/.tmux.conf に以下の設定が必要でした。

set -g pane-border-ascii on
